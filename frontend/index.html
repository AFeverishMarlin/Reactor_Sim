OPERATING GUIDE
OUTPUT = thermal% × 3200 MW × turbine efficiency
TURBINE EFF ≈ 31.5% clean steam → drops as void rises
0% void → 31.5% eff (1008 MWe max)  |  50% void → ~26% eff (~830 MWe)
MORE RODS OUT → ↑ flux → ↑ heat → ↑ temp → ↑ void → ↑ power
Target: 65–80% avg withdrawal · more = higher temp risk
MORE FLOW → ↓ temp → ↓ void → ↑ turbine eff · less = dangerous
Target: 70–80% pump speed · below 40% = alarm + void spike
XENON absorbs neutrons — builds after power cut, clears at high flux
After a SCRAM: Xe peaks for ~20s — restart needs all rods out
SWEET SPOT — 65–80% rods, 4×75% flow, T 280–305°C → 600–820 MWe
